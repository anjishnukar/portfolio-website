# Portfolio Website

A simple React portfolio starter built with Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Structure

- `index.html` — app entry point
- `src/main.jsx` — React bootstrap
- `src/App.jsx` — portfolio layout
- `src/index.css` — styles
